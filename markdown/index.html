❐ 제목(header)

#을 사용하여 제목을 추가할 수 있습니다.
#은 1~6개까지 지원합니다.

[입력]

markdown
# h1
## h2
### h3
#### h4
##### h5
###### h6

[결과]

h1

h2

h3

h4

h5
h6




❐ 줄 바꿈

기본적으로 줄을 바꿔 작성했어도 줄 바꿈이 되지 않습니다.

[입력]

markdown
첫 번째 문단입니다.
두 번째 문단입니다.

[결과]
첫 번째 문단입니다. 두 번째 문단입니다.



문단 끝에 공백 2개를 넣으면 줄 바꿈을 할 수 있습니다.

[입력]

markdown
첫 번째 문단입니다.(공백)(공백)  
두 번째 문단입니다.

[결과]
첫 번째 문단입니다.
두 번째 문단입니다.



br 태그를 사용하여 빈줄을 추가할 수 있습니다.

[입력]

markdown
첫 번째 문단입니다.
<br></br>
두 번째 문단입니다.

[결과]
첫 번째 문단입니다.


두 번째 문단입니다.



❐ 텍스트 강조

[입력]

markdown
**Bold**
__Bold__
*Italic*
_Italic_
~~Cancel~~
<u>밑줄</u>
<span style="color:red">빨간 글씨</span>
<mark>형광펜</mark>

[결과]
Bold
Bold
Italic
Italic
Cancel

밑줄
빨간 글씨
형광펜



❐ 인용문

[입력]

markdown
>인용문

[결과]

인용문





❐ 수평선

*, -, _을 3개 이상 사용하여 수평선을 추가할 수 있습니다.

[입력]

markdown
***
---
___

[결과]





❐ 순서 있는 목록

숫자와 .을 사용하여 순서 있는 목록을 추가할 수 있습니다.

[입력]

markdown
1. 첫 번째 목록
2. 두 번째 목록
3. 세 번째 목록

[결과]

  1. 첫 번째 목록
  2. 두 번째 목록
  3. 세 번째 목록



❐ 순서 없는 목록

-, +, * 을 사용하여 순서 없는 목록을 추가할 수 있습니다.

[입력]

markdown
- 순서 없는 목록 1
- 순서 없는 목록 2
- 순서 없는 목록 3
+ 순서 없는 목록 1
+ 순서 없는 목록 2
* 순서 없는 목록 1
* 순서 없는 목록 2

[결과]

  • 순서 없는 목록 1
  • 순서 없는 목록 2
  • 순서 없는 목록 3
  • 순서 없는 목록 1
  • 순서 없는 목록 2
  • 순서 없는 목록 1
  • 순서 없는 목록 2

tab을 사용하여 들여쓰기를 할 수 있습니다.

[입력]

markdown
- 순서 없는 목록 1
  - 순서 없는 목록 1-1
    - 순서 없는 목록 1-2

[결과]

  • 순서 없는 목록 1
    • 순서 없는 목록 1-1
      • 순서 없는 목록 1-2




❐ 이미지

[입력]

markdown
![](/assets/photo.jpg)

[결과]



뒤에 url을 추가로 넣으면 이미지에 링크를 추가할 수 있습니다.

[입력]

markdown
[![](/assets/photo.jpg)](https://github.com/zdlath)

[결과]





❐ 링크

[입력]

markdown
구글 홈페이지 : https://google.com

[결과]
구글 홈페이지 : https://google.com



특정 문구에 링크를 추가하고 싶은 경우

[입력]

markdown
[구글 홈페이지](https://google.com) 입니다.

[결과]
구글 홈페이지 입니다.



특정 문구 링크에 마우스를 올려놓으면 말풍선이 나오게 하고 싶은 경우

[입력]

markdown
[구글 홈페이지](https://google.com, "마우스를 올려놓으면 말풍선이 나옵니다.") 입니다.

[결과]
구글 홈페이지 입니다.



특정 URL을 정의해두고 해당 URL을 참조하여 링크를 추가하고 싶은 경우

[입력]

markdown
[구글 홈페이지][Google URL] 입니다.  
This is [Google HomePage][Google URL]


[Google URL]: https://www.google.com

[결과]
구글 홈페이지 입니다.
This is Google HomePage





❐ 인라인 코드

백틱(｀)을 사용하여 인라인 코드를 작성할 수 있습니다.

[입력]

markdown
`인라인 코드`입니다.

[결과]
인라인 코드입니다.





❐ 코드 블럭

``` 사이에 코드를 넣어 코드 블럭을 추가할 수 있습니다. ``` 시작 줄 옆에 프로그래밍 언어를 넣어주면 해당 언어 문법에 맞게 보여줍니다.

[입력]

markdown
```javascriptㅤ
let sumNumbers = (firstNum, lastNum) => {  
   return firstNum + lastNum;  
};  
sumNumbers(100, 200);  
```ㅤ

[결과]

javascript
let sumNumbers = (firstNum, lastNum) => {
  return firstNum + lastNum;
};
sumNumbers(100, 200);




❐ 표

|와 ---을 이용하여 표를 추가할 수 있습니다.

[입력]

markdown
|    |영어|수학|국어|
|----|---|---|---|
|임소희|60 |100|75 |
|김충기|80 |90 |85 |

[결과]

영어수학국어
임소희6010075
김충기809085


셀을 정렬하고 싶은 경우
:을 왼쪽에 추가하면 왼쪽 정렬ㅤㅤㅤㅤex) |:---------|
:을 양쪽에 추가하면 가운데 정렬ㅤㅤㅤex) |:---------:|
:을 오른쪽에 추가하면 오른쪽 정렬ㅤㅤex) |---------:|

[입력]

markdown
|왼 쪽 정 렬|가 운 데 정 렬|오 른 쪽 정 렬|
|:--------|:---------:|----------:|
|왼쪽1     |가운데1      |오른쪽1      |
|왼쪽2     |가운데2      |오른쪽2      |

[결과]

왼 쪽 정 렬가 운 데 정 렬오 른 쪽 정 렬
왼쪽1가운데1오른쪽1
왼쪽2가운데2오른쪽2




❐ 접기

[입력]

markdown
<details>
<summary>자세히 보기</summary>
자세한 설명입니다.
</details>

[결과]

자세히 보기자세한 설명입니다.




❐ 각주

[입력]

markdown
각주 1번 입니다.[^1]  
각주 2번 입니다.[^2]


[^1]: 각주 1번에 대한 설명
[^2]: 각주 2번에 대한 설명

[결과]
각주 1번 입니다.1
각주 2번 입니다.2
